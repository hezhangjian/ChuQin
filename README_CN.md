# ChuQin

![License](https://img.shields.io/badge/开源许可证-Apache2.0-green)

[English](README.md) | 简体中文

*ChuQin* 是一个基于 PARA 方法构建的本地优先桌面工作空间，面向个人工作者。

它围绕用户自有根目录提供沉浸式工作体验，让项目、资源与归档内容能够以一致方式组织，并通过集成工具区进行操作。

GUI 是 ChuQin 的主要交付形态，用于在可见上下文中管理 PARA 工作空间并复用各类工具能力。

CLI 提供桌面应用之外的命令行能力。
