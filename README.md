# ChuQin

![License](https://img.shields.io/badge/license-Apache2.0-green)

English | [简体中文](README_CN.md)

**ChuQin** is a local-first personal workspace assistant built around a user-owned filesystem root. It helps people manage
files, projects, knowledge, conversations, apps, and repeatable workflows in one desktop workbench.

It provides an immersive main-area experience centered on that root directory, where files, projects, knowledge, and
workflows can stay organized, visible, and actionable through an integrated tool area.

The GUI is the primary delivery surface of ChuQin. It is designed to let users work inside their personal workspace with
visible context and reusable tools. CLI provides command-line capabilities outside the desktop application.
